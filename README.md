# Stargaze Ami Hatch Chrome Plugin

This Chrome extension/plugin shows the Andromaverse collection as hatched Ami on the Stargaze marketplace and within your wallet.

> It still shows the original egg image on the individual token pages.

## Installation

Visit chrome://extensions and enable developer mode.

Click "Load Unpacked"

Select the "sg-ami-hatch" directory and load.

## Usage

Next time you visit Stargaze Andromaverse marketplace page or your wallet page, any Andromaverse eggs should load as hatched Ami.
